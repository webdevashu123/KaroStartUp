## India's Premier Startup Pitch Platform

**Karo Pitch** connects promising startups from across Bharat – especially Tier-2 & Tier-3 cities – with top investors through curated pitch events and a startup discovery platform.

![Hero Screenshot](screenshots/hero.png)
*Karo Pitch Landing Page – Modern glassmorphism design with particle effects*

## ✨ Features

- 🎨 **Glassmorphism UI** – Stunning backdrop filters, gradients, and neumorphic shadows
- 🌙 **Dark/Light Theme Toggle** – Persistent via localStorage
- ⚡ **Smooth Animations** – Scroll-triggered reveals, counter animations, rocket launch effects
- 🎭 **Interactive Hero** – Canvas particles, mouse parallax, floating rocket
- 📱 **Fully Responsive** – Mobile-first design, optimized for all devices
- 🚀 **Performance Optimized** – IntersectionObserver, requestAnimationFrame, lazy loading prep
- 📝 **Apply Modal** – Interactive form (demo submission)
- 📊 **Live Counters & Stats** – Animated metrics
- ♿ **Accessibility Ready** – Reduced motion support, keyboard navigation

## 🛠️ Tech Stack

| Category | Technologies |
|----------|--------------|
| **Frontend** | HTML5, CSS3 (Grid/Flexbox, Custom Properties, Backdrop Filter), Vanilla JavaScript ES6+ |
| **Styling** | Glassmorphism, CSS Animations, Gradients, CSS Variables |
| **Assets** | Font Awesome (CDN), Google Fonts (Manrope/Sora), Custom Hero Image |
| **Performance** | Canvas Particles, Intersection Observer, Smooth Scrolling |

**Zero Dependencies** – Pure vanilla code, no frameworks or build tools!

## 🎯 Customization

1. **Update Content**: Edit HTML sections (stats, startups, investors)
2. **Change Colors**: Modify CSS custom properties in `:root`
3. **Hero Image**: Replace `image/image.jpg`
4. **Real Stats**: Update `data-target` in counter elements
5. **API Integration**: Connect apply form to backend (Formspree, EmailJS)
6. **More Pages**: Add features to existing HTML templates
